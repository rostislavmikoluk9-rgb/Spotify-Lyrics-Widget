# 🎵 Spotify Synced Lyrics Widget

Минималистичный кроссплатформенный виджет для отображения текстов песен Spotify в реальном времени.

## 🛠 Установка и запуск

### **Windows**
1. Убедитесь, что установлен **Python 3.10+**.
2. Откройте терминал (PowerShell или CMD) в папке с проектом.
3. Запустите автоматический установщик:
   ```bash
   python install.py
   ```
4. После завершения запустите приложение:
   ```bash
   python main.py
   ```

### **Linux (Arch)**
1. Установите необходимую системную зависимость:
   ```bash
   sudo pacman -S playerctl
   ```
2. Запустите автоматический установщик зависимостей Python:
   ```bash
   python install.py
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

---

## ⚙️ Настройка (`config.json`)

Файл конфигурации создается автоматически при первом запуске в папке с приложением. Основные параметры:

| Параметр | Описание |
| :--- | :--- |
| **font_name** | Название шрифта (например, "Segoe UI" или "JetBrains Mono") |
| **font_size** | Размер текста в пикселях |
| **text_color** | Цвет текста (HEX или стандартные названия цветов) |
| **offset** | Калибровка задержки |
| **x, y** | Координаты окна на рабочем столе |
| **shadow_blur** | Радиус размытия тени под текстом |

---

## 🖥 Системные требования

* **Windows:** 10/11 (требуется библиотека `winsdk`).
* **Linux:** Наличие `playerctl` и поддержка `xcb` (тестировалось на KDE Plasma).
* **Spotify:** Должен быть запущен официальный десктопный клиент.

## 📁 Структура проекта
* `main.py` — основной файл приложения и логика синхронизации.
* `settings.py` — окно настроек.
* `install.py` — скрипт автоматической установки зависимостей.
* `lyrics_cache/` — папка с сохраненными текстами песен в формате `.lrc`.
